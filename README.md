# 格狀鉸鏈設計推論系統 Lattice Hinge Designer

雷射切割彎木（laser kerf bending / lattice hinge）的參數化設計計算器。
輸入材料與幾何參數,即時推算最少連桿數、彎折半徑與工作應力,並可匯出雷射切割用 SVG 檔。

**線上使用**: https://henrychao521.github.io/lattice-hinge-designer/
**開發對話紀錄**: https://henrychao521.github.io/lattice-hinge-designer/dev-log.html （8 輪對話 Rev.A→H 的完整開發歷程）
**範例檔**: [`examples/laser33050mm.dxf`](examples/laser33050mm.dxf) — Onshape 匯出的 330×50mm 鉸鏈件,可直接餵給「DXF 逆向分析」試用

## 功能

- 六種材料預設值（壓克力、樺木合板、MDF、實木、POM、PP）,G 與 τ 可手動覆寫
- 依 Fenner 扭轉連桿模型計算:`n ≥ (β/α)·Θ·G·b/(τ_allow·l)`
- 幾何干涉檢查（單刀切縫條件:`w·cosθ + t·sinθ ≤ w + k`）
- 設計應力上限滑桿（靜態彎折 vs 反覆開合的疲勞裕度）
- 六種切割圖樣即時預覽:直線格狀（扭轉式）、波浪（彎曲式）、十字（混合式）、長孔/蜂巢（挖除式）、斜向 45°、Auxetic 旋轉方塊（雙曲面用,含伸展率公式 L(θ)/L₀ = cos(θ/2)+sin(θ/2)）
- 3D 彎折模擬:拖曳旋轉視角、彎折程度滑桿,切割線貼合曲面,調參即時更新（純 canvas,零外部函式庫）
- 變半徑設計器:拖曳控制點畫出目標輪廓曲線,系統依局部曲率自動放置漸變間距切線列（曲率大處加密、直線段不切）,含最小半徑可行性檢查與 SVG 匯出
- 匯出雷射切割 SVG（紅線 0.1mm 切割線,內嵌參數註解可追溯）、DXF（R12,CUT/OUTLINE 圖層）與參數報告 txt
- DXF 逆向分析:匯入外部 DXF 自動偵測格狀參數、推算可行性（可達彎角/最小半徑）、淹水填充切斷檢查與製造品質警告
- 教學用逐步推導展開面板,適合課堂展示計算流程

## 技術說明

純靜態單檔 HTML + vanilla JS,無後端、無建置流程,直接以 GitHub Pages 部署。

## 理論來源

- P. Fenner, *Laser-cut Lattice Living Hinges* 系列 (Deferred Procrastination, 2011–2012) — 扭轉連桿模型與公式推導
- A. Porterfield, *Curved Laser Bent Wood* (Instructables, 2014) — 曲面彎折與圖樣實驗
- Trotec Laser, *Cutting technique for bending applications* — 材料別 kerf 圖樣建議
- J. Grima & K. Evans, rotating squares 拉脹模型 — Auxetic 圖樣伸展率公式

## 注意事項

木質材料的剪切模數 G 與降伏應力 τ 受樹種、纖維方向、含水率影響極大,
表列值僅供初估,實際設計請以試片實測校正。

## 修訂紀錄

- **Rev.H (2026-07-07)** — 3D 彎折模擬:
  - 純手寫 canvas 3D 渲染（零外部函式庫）:平板—圓弧—平板可展曲面,依推論結果 W/R/Θ 即時成形
  - 拖曳旋轉視角、彎折程度滑桿（0→Θ 展開動畫）,切割線同步貼合曲面
  - 調整任何參數（材料/機台/幾何/圖樣）3D 圖即時更新;混合式圖樣以圓柱面近似並標注
- **Rev.G (2026-07-07)** — DXF 佈線:在匯入圖上加入切割線圖
  - 匯入 DXF 後可直接佈置鉸鏈帶:選圖樣（六種皆可）與方向（縱/橫）、起點、延伸寬度、段數、段間距、留邊
  - 即時預覽（藍=原圖、紅=新增切線）,扭轉式圖樣即時推算每段/合計可彎角度
  - 一鍵切斷檢查（合成後整體淹水填充）;超出工件範圍或段重疊會警告
  - 合成輸出 DXF（ORIGINAL/HINGE 分層）與 SVG（藍/紅分色）
- **Rev.F (2026-07-07)** — DXF 逆向分析（可行性檢查）:
  - 匯入外部 DXF（Onshape/Illustrator/Fusion 等,LINE 實體）,瀏覽器本機解析不上傳
  - 自動偵測格狀參數:切線欄數、行距、切段長、junction、交錯重疊（有效連桿長）
  - 依目前材料/機台設定推算可達總彎角與最小中心半徑,調參數即時重算
  - 像素化淹水填充切斷檢查（0.1mm）;另檢查 junction 過小、邊軌過窄、切線超出外框、重複/壓線（二次燒切）、非 mm 單位
- **Rev.E (2026-07-07)** — DXF 匯出＋Auxetic 韌帶警告:
  - 新增 DXF 匯出（R12 ASCII、LINE 實體、CUT/OUTLINE 圖層、$INSUNITS=mm）:主圖樣與變半徑各一鍵,波浪貝茲與長孔圓弧自動展成折線,RDWorks/LightBurn/AutoCAD 皆可讀
  - Auxetic 旋轉方塊:標明每個角隅節點必有一條切線穿過、鉸鏈被剖成兩條韌帶（有效寬度 = (h−k)/2 每側）;h ≤ k 時韌帶全失、方塊散落,新增紅色警告;韌帶 < 0.3mm 黃色警告
- **Rev.D (2026-07-07)** — 雷射切割機變因＋防切斷保護:
  - 新增切割機選單（CO₂ 60–80W / CO₂ 30–40W / 二極體 / 自訂）,依機型×材料自動帶入建議 kerf 與切邊熱影響折減 η（τ_allow = τ_y × 上限% × η）,不適合的機材組合會警告
  - 新增 kerf 校正試片 SVG 匯出（20mm 方塊量測 kerf ＋ 迷你鉸鏈試彎段）
  - **修正切斷 bug**:H 小於切段長 2l+gap 時,相位對齊的切線會單段貫穿整個鉸鏈長、把板材切成兩半;現繪圖時自動在中央保留 junction,並顯示警告（變半徑設計器與校正試片同步保護）
  - 十字圖樣垂直臂夾限於板內,不再切出上下邊緣
- **Rev.C (2026-07-07)** — v2 功能合併＋內容修正:
  - 新增斜向 45°、Auxetic 旋轉方塊兩種圖樣與變半徑設計器（曲率自適應切線佈列）
  - 重新套用 Rev.B 全部修正（v2 由 v1 衍生,原修正曾被覆蓋退回）
  - 變半徑設計器:每列容許彎角改為 min(應力限制, 幾何干涉限制),與主計算一致;首列改放在彎曲起點（受扭區在列與列之間,原演算法整體少一列）
- **Rev.B (2026-07-07)** — 內容修正與優化:
  - 直線/波浪圖樣切割線改為 n+1 條（m 條交錯切線僅形成 m−1 個串聯扭轉連桿,原 n−1 條實際只有 n−2 個有效連桿）
  - 切段長改為 `2l+gap`,使相鄰列切縫重疊區（實際受扭長度）恰等於公式假設的連桿長 l（原本短少 gap/2,應力低估約兩成）
  - 十字圖樣間距改為臂長 + junction,修正原間距過密導致同列水平切線相連、板材直接斷開的問題
  - 幾何干涉角改求「第一個」違反角（投影寬度非單調,原二分法在 w≫t 時失效）
  - 新增內彎半徑小於板厚警告、輸入欄位清空防呆;雷射 SVG 內嵌參數註解便於追溯
- **Rev.A** — 初版
